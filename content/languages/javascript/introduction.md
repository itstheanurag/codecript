JavaScript is the language of the web — the only programming language that runs natively in every browser.

## Why Learn JavaScript

- Powers every interactive website
- Full-stack capable with Node.js
- Massive ecosystem (npm has over 2 million packages)
- Gateway to React, Vue, Angular, and more

## History

Created by Brendan Eich in 1995 in just 10 days, JavaScript has evolved from a simple scripting language into one of the most important programming languages in the world. The ECMAScript standard (ES6/ES2015 and beyond) brought modern features that transformed how we write JavaScript.

## Running JavaScript

```javascript
// In the browser console
console.log("Hello, World!");

// With Node.js
// $ node script.js
```

This section covers JavaScript from the ground up — variables, functions, objects, async programming, and modern ES6+ features.
