Python is a high-level, interpreted programming language known for its readability and simplicity.

## Zen of Python

> Beautiful is better than ugly.
> Explicit is better than implicit.
> Simple is better than complex.
> Readability counts.

## Hello World

```python
print("Hello, CodeCript!")
```

## Key Characteristics

- **Interpreted**: Code is executed line-by-line.
- **Dynamically Typed**: You don't declare types; they are assigned at runtime.
- **Indentation Primary**: Uses indentation instead of braces `{}` to define blocks.
